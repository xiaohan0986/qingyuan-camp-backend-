```javascript

})
```

## 🚨 常见错误及解决方案

### ❌ 错误 1: 401 未授权
**症状**: API 返回 `{"code":401,"message":"请先登录"}`  
**原因**: Session 未传递到 API  
**解决**: 在 fetch 请求中添加 `credentials: 'same-origin'`

### ❌ 错误 2: 数据库连接失败
**症状**: 数据库连接错误  
**原因**: 用户名/密码错误  
**解决**: 检查 `config/database_platform.php` 配置

### ❌ 错误 3: 列表为空
**症状**: 用户列表显示"暂无数据"  
**原因**: 数据库中无数据  
**解决**: 导入 `database/platform_users.sql`
