# 客户管理全新升级

## 📋 功能说明

客户管理表单全面升级，支持完整的信息采集、资料上传和进度追踪。

### ✅ 核心功能

#### 一、基础信息（必填）
- 客户名称（从用户管理选择）
- 年龄
- 电话
- 学历
- 邮箱
- 微信
- 国家
- 岗位
- 签证类型
- 来源
- 销售顾问

#### 二、基本情况（非必填）
- 婚姻状况
- 子女状况
- 工作状况
- 流水状况
- 社保状况
- 技能
- 籍贯
- 车辆
- 房产
- 财务状况

#### 三、所需资料（按钮式上传）
- 无犯罪证明
- 工资流水
- 社保证明
- 工作证明
- 签证申请表
- 身份证
- 结婚证
- 户口本
- 机动车证
- 房产证
- 护照
- 2 寸白底照片
- 其他

**特性：**
- 点击按钮生成文件上传框
- 支持图片和 PDF 文件
- 每个资料类型只能上传一个文件（"其他"可多个）
- 上传后显示文件预览

#### 四、办理进度追踪
- 输入正在办理的项目
- 点击"更新进度"添加记录
- 自动保留历史进度
- 显示时间戳和进度内容
- 示例：2026-3-19-开始办理/2026-3-22-提交客户基础信息/2026-3-28-以提交移民局审核资料

### 新建 customer_documents 表

```sql
CREATE TABLE `customer_documents` (
    `id` INT AUTO_INCREMENT PRIMARY KEY,
    `customer_id` INT NOT NULL COMMENT '客户 ID',
    `doc_type` VARCHAR(50) NOT NULL COMMENT '资料类型',
    `file_path` VARCHAR(500) NOT NULL COMMENT '文件路径',
    `file_name` VARCHAR(200) NOT NULL COMMENT '文件名',
    `file_size` INT DEFAULT 0 COMMENT '文件大小 (字节)',
    `uploaded_at` DATETIME DEFAULT CURRENT_TIMESTAMP,
    `uploaded_by` INT DEFAULT NULL COMMENT '上传人 ID',
    INDEX `idx_customer_id` (`customer_id`),
    INDEX `idx_doc_type` (`doc_type`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 新建 customer_progress 表

```sql
CREATE TABLE `customer_progress` (
    `id` INT AUTO_INCREMENT PRIMARY KEY,
    `customer_id` INT NOT NULL COMMENT '客户 ID',
    `progress_text` TEXT NOT NULL COMMENT '进度内容',
    `created_at` DATETIME DEFAULT CURRENT_TIMESTAMP,
    `created_by` INT DEFAULT NULL COMMENT '操作人 ID',
    INDEX `idx_customer_id` (`customer_id`),
    INDEX `idx_created_at` (`created_at`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

## 🎯 API 变更

### api/customer.php - 新增接口

#### 获取资料列表
```
GET /api/customer.php?action=get_documents&customer_id=1
```

#### 上传资料
```
POST /api/customer.php?action=upload_document
FormData: customer_id, doc_type, file
```

```
POST /api/customer.php?action=delete_document
FormData: id
```

#### 获取进度历史
```
GET /api/customer.php?action=get_progress&customer_id=1
```

#### 添加进度
```
POST /api/customer.php?action=add_progress
FormData: customer_id, progress_text
```

## 💡 使用流程

### 新增客户

1. 点击"➕ 新增客户"按钮
2. **填写基础信息**（所有带*的字段）
   - 选择用户
   - 填写年龄、电话、学历等
3. **填写基本情况**（可选）
   - 婚姻、子女、工作状况等
4. **添加所需资料**
   - 点击资料类型按钮
   - 生成文件上传框
   - 选择图片/PDF 文件上传
5. **记录办理进度**
   - 输入当前办理项目
   - 点击"更新进度"
6. 点击"保存"按钮

### 编辑客户

1. 点击客户列表中的"📝"编辑按钮
2. 表单自动填充所有信息
3. 修改需要更新的字段
4. 可添加/删除资料
5. 可添加新的进度记录
6. 点击"保存"按钮

### 查看进度历史

编辑客户时，在"办理进度"区域可以看到：
```
2026-04-08 23:00:00 - 开始办理
2026-04-08 23:05:00 - 提交客户基础信息
2026-04-08 23:10:00 - 提交移民局审核资料
```

## ⚠️ 注意事项

### 必填验证
- 基础信息所有字段必填
- 前端验证 + 后端验证
- 未填写时无法保存

### 文件上传
- 支持格式：JPG, PNG, GIF, PDF
- 文件大小：最大 10MB
- 存储路径：`memory/documents/customer_{id}/`
- 每个类型只能一个文件（"其他"除外）

### 进度记录
- 自动添加时间戳
- 按时间倒序显示（最新在前）
- 不可删除或修改历史记录
