# 操作日志系统

## 功能说明
实时记录用户在系统中的所有关键操作，包括菜单点击、数据创建、修改、删除等。

## 日志格式
```
[2026-04-08 21:42:50] [用户:admin] [IP:127.0.0.1] 点击菜单 - 岗位管理
[2026-04-08 21:43:50] [用户:admin] [IP:127.0.0.1] 创建岗位 - 岗位名称：新加坡中文教师
[2026-04-08 21:44:50] [用户:admin] [IP:127.0.0.1] 保存数据 - 岗位 ID: 15
```

## 文件结构
```
qianwutong/
├── includes/
│   └── OperationLogger.php      # 日志记录器类
├── api/
│   └── log_operation.php         # 日志记录 API
├── admin/
│   ├── includes/
│   │   └── header.php            # 集成日志记录（菜单点击）
│   ├── diagnosis/
│   │   └── operation_log.php     # 日志查看页面
│   └── position.php              # 岗位管理（已集成日志）
└── memory/
    └── logs/
        └── operation_YYYY-MM-DD.log  # 每日日志文件
```

## 记录的操作类型

### 1. 菜单点击
- 自动记录所有侧边栏菜单点击
- 记录内容：菜单名称

### 2. 数据操作
- **创建**：创建新岗位、新客户、新用户等
- **更新**：修改现有数据
- **删除**：删除数据
- **状态切换**：上架/下架、启用/禁用等

### 3. 其他操作
- 登录/登出
- 数据导出
- 批量操作

## 使用方法

### 前端 JavaScript
```javascript
// 记录操作
fetch('../api/log_operation.php', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
        action: '操作类型',
        detail: '操作详情',
        data: {额外数据}
    })
});
```

### 后端 PHP
```php
require_once 'includes/OperationLogger.php';

$logger = new OperationLogger();
$logger->log('操作类型', '操作详情', ['key' => 'value']);
```

## 查看日志

### 1. 访问日志页面
URL: `http://localhost/qianwutong/admin/diagnosis/operation_log.php`

### 2. 功能
- 查看今日操作
- 查看近 7 天操作
- 查看近 30 天操作
- 按操作类型统计
- 实时刷新

### 3. 日志内容
- 时间戳（精确到秒）
- 操作用户
- 操作 IP
- 操作类型
- 操作详情
- 额外数据

## 日志文件

### 存储位置
`memory/logs/operation_YYYY-MM-DD.log`

### 文件格式
每行一条记录，包含完整操作信息。

### 日志轮转
- 每天创建一个新文件
- 自动清理 30 天前的日志（需手动实现）
