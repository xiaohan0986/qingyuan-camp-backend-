## 💡 使用说明

### 查看日志
- 访问：`http://localhost/qianwutong/admin/diagnosis/operation_log.php`
- 支持：今日 / 近 7 天 / 近 30 天
- 格式：纯文本终端风格

### 日志文件位置
```
D:\phpstudy_pro\WWW\qianwutong\memory\logs\operation_2026-04-08.log
```

### 日志格式
```
[时间戳] [用户：用户名] [IP:IP 地址] 操作类型 - 操作详情 [JSON 数据]
```
