# 操作日志功能

## 功能说明
在诊断工具中添加了操作日志查看器，可以查看近 30 天内对项目的所有文件修改操作。

## 访问路径
- **URL**: `http://localhost/qianwutong/admin/diagnosis/operation_log.php`
- **入口**: 诊断工具首页 → 高级调试 → 📋 操作日志

## 功能特点
1. **时间范围**: 显示近 30 天的所有文件修改
2. **文件类型**: 支持 PHP、JS、CSS、HTML、MD、SQL、JSON 等
3. **分组显示**: 按日期分组，今天/昨天/具体日期
4. **统计信息**: 
   - 总修改文件数
   - PHP 文件数
   - CSS 文件数
   - JS 文件数
   - 文档文件数
5. **详细信息**: 
   - 文件路径
   - 修改时间（精确到秒）
   - 文件大小

## 技术实现
- 使用 PHP `RecursiveIteratorIterator` 递归遍历项目目录
- 按文件扩展名分类显示不同颜色图标
- 支持 hover 动画效果
- 响应式布局

## 使用场景
1. 追踪最近的代码修改
2. 排查问题时间线
3. 审计开发进度
4. 回顾工作内容
