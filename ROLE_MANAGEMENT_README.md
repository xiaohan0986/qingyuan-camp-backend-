# 角色管理系统安装指南

## 📋 功能概述

角色管理系统允许您：
- ✅ 创建和管理多个角色（如超级管理员、系统管理员、运营人员、客服人员等）
- ✅ 为每个角色配置细粒度的操作权限（查看、创建、编辑、删除等）
- ✅ 创建可登录后台的用户并分配角色
- ✅ 修改用户名、密码
- ✅ 移除用户或更改用户角色

## 🚀 安装步骤

### 1. 执行数据库安装

在浏览器中访问：
```
http://你的域名/admin/install_roles.php
```

这将自动创建：
- `roles` 表（角色表）
- `role_permissions` 表（角色权限关联表）
- `users.role_id` 字段（用户角色关联字段）
- 4 个默认角色和权限配置

### 2. 设置超级管理员

安装完成后，需要将当前管理员账户设置为超级管理员：

```sql
-- 将用户 ID=1 的管理员设置为超级管理员（角色 ID=1）
UPDATE users SET role_id = 1 WHERE id = 1;
```

或者在 phpMyAdmin 中手动修改 `users` 表的 `role_id` 字段为 `1`。

### 3. 访问角色管理

安装完成后，点击右上角头像 → 系统设置 → **角色管理**

## 📊 默认角色

系统预置了 4 个角色：

| 角色名称 | 角色标识 | 权限说明 |
|---------|---------|---------|
| 超级管理员 | super_admin | 拥有所有权限 |
| 系统管理员 | admin | 拥有系统管理权限 |
| 运营人员 | operator | 拥有运营相关权限 |
| 客服人员 | service | 拥有客服相关权限 |

## 🔧 使用说明

### 创建角色

1. 点击"➕ 新增角色"
2. 填写角色名称和角色标识（英文）
3. 配置权限（可选择模块和具体操作）
4. 点击保存

### 管理角色用户

1. 在角色列表中点击"👥 用户"
2. 可以添加新用户（用户名 + 密码）
3. 可以修改用户角色或移除用户

### 修改用户信息

在角色用户管理弹窗中：
- 点击"修改"可以更改用户的角色
- 点击"移除"可以将用户从当前角色移除

## ⚠️ 注意事项

1. **超级管理员角色不能删除**
2. 删除角色时，该角色下的用户会被设置为无角色状态
3. 只有超级管理员可以管理角色
4. 建议先创建测试角色进行测试

## 🔐 权限配置说明

权限配置采用 JSON 格式，支持两种模式：

### 模式 1：拥有模块所有权限
```json
{
  "customer": true
}
```
表示拥有客户管理的所有权限（查看、创建、编辑、删除等）

### 模式 2：细粒度权限
```json
{
  "customer": {
    "view": true,
    "edit": true
  }
}
```
表示只拥有客户管理的查看和编辑权限

### 模式 3：超级管理员
```json
{
  "all": true
}
```
表示拥有所有权限
