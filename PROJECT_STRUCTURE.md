# 签务通后台管理系统 - 项目结构

```
qianwutong/

│   ├── css/                  # 样式文件
│   │   ├── common.css       # 公共样式
│   │   ├── login.css        # 登录页样式
│   │   ├── dashboard.css    # 数据大屏样式
│   │   └── position.css     # 岗位管理样式
│   ├── images/               # 图片资源
│   │   ├── logo.svg         # Logo
│   │   └── avatar-placeholder.svg  # 头像占位图
│   ├── js/                   # JavaScript 文件
│   ├── index.php            # 登录页
│   └── dashboard.php        # 数据大屏
├── api/                      # API 接口
│   ├── auth.php             # 认证 API
│   ├── dashboard.php        # 数据大屏 API
│   ├── position.php         # 岗位管理 API
│   ├── customer.php         # 客户管理 API
│   └── marketing.php        # 营销获客 API
├── config/                   # 配置文件

│   └── config.php           # 系统配置
├── includes/                 # 公共库
│   ├── Database.php         # 数据库类
│   └── functions.php        # 公共函数
├── uploads/                  # 上传文件
│   ├── positions/           # 岗位图片
│   └── avatars/             # 用户头像

    └── DEPLOY.md            # 部署文档
```

## 🎨 样式文件说明

### common.css
- 全局样式（重置、字体、颜色）
- 表单样式（输入框、按钮）
- 布局样式（侧边栏、主内容区）
- 组件样式（卡片、表格、分页、模态框）
- 响应式设计

### login.css
- 登录页面专用样式
- 渐变背景
- 登录框动画
- 错误提示动画

### dashboard.css
- 数据大屏页面样式
- 统计卡片网格
- 图表卡片
- 访问趋势可视化

### position.css
- 岗位管理页面样式
- 筛选区域
- 操作栏
- 表单模态框

## 📊 已完成功能
